## 什么是GNU/Linux
- GMU是一套操作系统，是一组供用户使用的软件程序：
  - GNU名字的含义是一个递归缩写，代表“GNU's Not Unix”（GNU不是Unix）。这个名称由自由软件基金会（Free Software Foundation, FSF）的创始人理查德·斯托曼（Richard Stallman）提出，旨在创建一个完全自由的Unix兼容操作系统，目标是开发一个自由的操作系统，强调用户在使用软件时应享有自由，包括运行、复制、分发、学习、修改和改进软件的自由。
  - GNU操作系统由许多自由软件组成，这些软件遵循GNU通用公共许可证（GPL），这是一种广泛使用的自由软件许可证。因此GNU不仅是一个操作系统，它还包含大量的自由软件包，如编译器GCC、文本编辑器Emacs等，这些软件都是由GNU项目或第三方发布的。通过这些软件，用户可以自由地使用计算机而不必担心侵犯版权或受到商业软件的限制。
- Linux也是操作系统，但是最底层的软件，直接与硬件交互，称为操作系统的内核部分，负责管理硬件资源和为应用程序提供底层服务。
- Linux与GNU项目有着密切的关系：尽管Linux内核本身并不构成一个完整的操作系统，但其与GNU项目的结合使得它能够提供完整的系统功能，Linux内核负责硬件管理，而GNU工具链和软件则提供了用户界面、编程工具和其他应用程序。
- **GNU操作系统**通常与**Linux内核**结合使用，诞生了世界上最自由的 **GNU/Linux操作系统**。
- Linux发行版（如Ubuntu、Debian、Fedora等）通常被称为“Linux”，但实际上它们更准确地应该被称为“GNU/Linux”，因为它们依赖于GNU项目提供的大量软件。
